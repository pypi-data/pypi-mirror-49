oci-cli-rc
**********


Description
===========

Generates a oci_cli_rc file that can contain parameter default values
and other configuration information such as command aliases and
predefined queries.

This command will populate the file with some default aliases and
predefined queries.


Usage
=====

   oci setup oci-cli-rc [OPTIONS]


Options
=======


--file [filename]
-----------------

The file into which default aliases and predefined queries will be
loaded [default: ~/.oci/oci_cli_rc]


-?, -h, --help
--------------

For detailed help on any of these individual commands, enter <command>
--help.
