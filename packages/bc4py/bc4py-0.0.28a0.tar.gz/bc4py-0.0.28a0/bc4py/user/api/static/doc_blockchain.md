get mintcoin info
----
* Arguments
    1. mint_id   (numeric, optional, default=0)
* Request example
    * `curl -H "accept: application/json" -H "Content-Type: application/json" "127.0.0.1:3000/public/getmintinfo?mint_id=3414534983"`
* Response
```json
{
    "version": 4,
    "coin_id": 3414534983,

    "unit": "FC",
    "digit": 8,
```
